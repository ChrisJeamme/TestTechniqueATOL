# Test technique Atol

- Cloner le dépôt : `git clone https://github.com/ChrisJeamme/TestTechniqueATOL`
- `cd TestTechniqueATOL`

## Serveur
- Créer une database MongoDB "testatol" (ou modifier le .env)
- `cd Server`
- `npm i`
- `npm start`


## Application
- `cd Application`
- `npm i`
- `npm start`
